# bankist-app-tutorial-example

A modern, minimalistic banking interface originally created by Jonas Schmedtmann in his JavaScript course and recoded by me to deepen my understanding of DOM manipulation, events, arrays, dates, timers, and clean JavaScript architecture.

This project simulates basic banking operations such as transferring money, requesting loans, and viewing transaction history â€” all using pure JavaScript.

## Features
### Core Banking Functionalities
  + Login system (username + pin)<br />
  + View account balance<br />
  + Display transaction history (movements)<br />
  + Deposit & withdrawal simulation<br />
  + Money transfer between accounts<br />
  + Loan request simulation<br />
  + Account closure option<br />
